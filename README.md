## Getting Started

### Installing

To clone the project and get it started, please follow these steps:

1. Clone the repo:

```bash
git clone https://github.com/tnjensen/portfolio-1.git
```

2. Install the dependencies:

```
npm install
```

### Running

To see the site, please visit:
